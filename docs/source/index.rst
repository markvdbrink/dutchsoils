Introduction
============

Welcome to the documentation of DutchSoils! This package loads, formats and plots soil physical, textural and chemical data from the `Dutch Soil Map <https://www.wur.nl/nl/show/bodemkaart-van-nederland.htm>`_, the `BOFEK soil clustering <https://www.wur.nl/nl/show/Bodemfysische-Eenhedenkaart-BOFEK2020.htm>`_ and the hydraulic parameters from the `Staring series <https://research.wur.nl/en/publications/waterretentie-en-doorlatendheidskarakteristieken-van-boven-en-ond-5>`_.
